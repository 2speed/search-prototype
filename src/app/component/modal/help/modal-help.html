Search

To perform a search, simply enter a search term in the search field:



and click the button. Search results will be displayed in the table below the search field.


Search Syntax

Search text is parsed into a series of terms and operators. A term can be a single word  tcga, or a phrase enclosed with double quotes "tcga dcc" to search for all the words in the phrase in the same order.

Additional search syntax (wildcard, fuzzy, range, etc.) can be found here: Apache Lucene Query Syntax.


Typeahead/Auto-complete

As search terms are entered into the search field, similar terms may be suggested that can be used to complete the search text. Suggested terms are retrieved directly from the search index in real-time and are based on specific field values for a document type.


MAF
  • Hugo_Symbol
  • Tumor_Sample_Barcode
XML
  • disease_code
  • bcr_patient_barcode

To ignore suggested terms and enter further search terms press the escape Esc key.


JSON Output

The raw JSON output for each search request will provided in the panel labeled Raw JSON Response.
